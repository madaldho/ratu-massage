# ratu-massage

Landing page Astro untuk layanan massage panggilan Ratu Massage & Beauty di Bogor.

## Tech Stack

- Astro 5
- TypeScript
- CSS custom components

## Scripts

| Command | Keterangan |
| :------ | :--------- |
| `npm install` | Install dependency project |
| `npm run dev` | Menjalankan development server |
| `npm run build` | Build production ke folder `dist/` |
| `npm run preview` | Preview hasil build secara lokal |

## Struktur Utama

```text
src/
	components/
	layouts/
	pages/
	styles/
public/
	assets/
```

## Deployment

- GitHub repository: akan di-push sebagai `ratu-massage`
- Vercel project: `ratu-massage`
- Production URL: akan diisi setelah deploy Vercel selesai

## Catatan

Project ini sudah dioptimalkan untuk landing page spa/massage dengan layout responsif, CTA WhatsApp, dan komponen Astro terpisah per section.
